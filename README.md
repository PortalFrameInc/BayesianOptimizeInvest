L'objectif du projet est de déterminé une stratégie d'investissement optimal via ETF sur long terme via inférence bayésienne, méthode de Monte-Carlo et formule de Kelly.

## Définition des stratégies

Mono-core et Core-Satellite

## Modéliser les paramètres initiales

- μ : rendement annuel espéré
- σ : volatilité annuelle
- frais etf
- correlation inter-actifs

## Faire les simulation de Monte-Carlo

100000 trajectoires par stratégie

Calculer :

- Volatilité du portefeuille
- Espérance
- CAGR moyen
- CAGR médian
- P25 du CAGR
- P75 du CAGR
- DD75
- DD95
